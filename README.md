# NFP119-APP-Tasks
Application React de gestion de tâches

- ### Lancer le container

```Shell
docker-compose up -d
```

- ### Arrêter le container

```Shell
docker-compose down
```

- ### Ouvrir le container dans un terminal

```Shell
docker exec -it NFP-119-APP-Tasks-node sh
```

- ### Sortir du terminal

###### Dans le terminal du container
```Shell
exit
```
